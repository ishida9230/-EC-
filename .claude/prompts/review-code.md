Review the following code and check for issues.

## Review Checklist

### Security
- [ ] SQLインジェクションの危険がないか（プレースホルダー使用を確認）
- [ ] 外部入力をそのままSQLに使っていないか
- [ ] 機密情報（パスワード等）をレスポンスに含めていないか

### Performance
- [ ] N+1クエリが発生していないか
- [ ] 不要なデータを全件取得していないか（SELECT *の乱用）
- [ ] 在庫減算がトランザクションで保護されているか

### Architecture
- [ ] ファイル配置がプロジェクトルールに従っているか
- [ ] スコープ外機能（認証・決済等）を誤って実装していないか
- [ ] TypeScript型が適切に定義されているか
